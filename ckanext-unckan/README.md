# CKAN unckan extension

CKAN extension to track users activities on the site through the CKAN API.

## Requirements

Compatibility with core CKAN versions:

| CKAN version    | Compatible?   |
| --------------- | ------------- |
| 2.10            | Yes           |
| 2.11            | not tested    |
